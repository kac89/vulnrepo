FAQ

1. Browser support

Project is working on latest browsers:

Google Chrome, Mozilla Firefox, Microsoft Edge (chromium), iOS Safari/Chrome, Android Chrome

 2. How reports are stored?

All data are stored in browser (on local machine, nothing is send outside) using indexeddb and SessionStorage, reports by default are encrypted in AES.

 3. How to restore my encrypted report if I forgot my security key?

Use bruteforce method or better remember the security key, otherwise there is no other option to decrypt data.

4. Can I change report security key?

If you able to decrypt report where security key you want to change, yes!.

 5. How does displaying attachments in the HTML report works?

It's simple, all images are always displayed and *.txt files. All other file types will be displayed as a link.

 6. How to create PDF report?

Download HTML report, open and use 'print to pdf' feature in your browser (ctrl+p): Chrome/FireFox.

 7. What is the difference between the file with the (.VULN) & (.VULNR) extension?

(.VULN) - File contains only issues.

(.VULNR) - File contains full report details e.g.: issues, metadata, author etc...
