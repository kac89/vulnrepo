Accelerate vulnerability reporting and securely share findings with vendors — or use it as a private bug-bounty repository. The ideal tool for pentesters, researchers, and anyone getting started in IT security.

grid_view Feature highlights

lock Security-first

AES encryption runs entirely in the browser. No data ever leaves your machine by default — no backend, no telemetry, pure JS client.

share Encrypted sharing

Share reports with vendors via AES-encrypted links. Recipients get the data, not the key — unless you send it.

sim_card_download Multiple export formats

Export to HTML, TXT, JSON, Markdown, DOCX, or PDF. The JSON format unlocks custom integrations — LaTeX, CI pipelines, and more.

attach_file File attachments

Attach screenshots, videos, or scanner output. Each file is automatically SHA-256 checksummed on upload.

description Issue templates

Build your own templates or start from defaults. Import CVE, CWE, MITRE ATT&CK, or PCI DSS data in one click.

input Scanner imports

Import findings directly from Nmap, Nessus, Burp Suite, OpenVAS, Bugcrowd, and Trivy. Edit and manage after import.

history Changelog tracking

Every significant change is automatically logged and versioned in a built-in changelog — full audit trail, zero effort.

bug_report Bug tracker export

Push issues directly to Atlassian JIRA or export as JSON for any other system. Write your own integration with the open API.

photo_filter AI integration

Connect your own local LLM. Ask AI to generate report summaries, enrich findings, and surface insights — all without sending data externally.
