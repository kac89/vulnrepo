VULNRΞPO is a FREE Open Source project with end-to-end encryption by default, designed to speed up the creation of IT Security vulnerability reports and can be used as a security reports repository.
Accelerates vulnerability reporting, you can quickly and securely share the report with the vendor or use as a repository with vulnerabilities for bug bounty research! the ideal tool for the Pentester, Researcher or anyone starting out in IT Security!













list   Features list The list includes only some of the basic features
security Security Project use browser for encrypt/decrypt (AES) and store data in locally. Full confidentiality of data, end-to-end encryption, by default nothing is sent out. No backend system, only front-end technology, pure JS client.

share Share report You can share your report using AES encryption by default.

cloud_download HTML & TXT & JSON & MARKDOWN & DOCX & PDF You can download report in HTML, TXT, JSON, MARKDOWN, DOCX formats!. If you need PDF just 'print as PDF' html report or from DOCX. The JSON format opens up completely new possibilities for processing report data.

attach_file Attachments You can easly attach any file you want to. Screenshot, movie or scanner output in txt. Automatically doing checksum sha256 of attached file.
info Issues templates! Create own templates or use default. The use of templates greatly speeds up the work for pentester or security auditor. Import CVE, CWE, MITRE ATT&CK or PCI DSS data also possible.

import_export Import issues from security scanners Supported import from: Nmap, Nessus, Burp, OpenVAS, Bugcrowd, Trivy. After importing, easily manage and edit vulnerabilities.

track_changes Changelog All important changes in report will be logged in to changelog and update to the next version of the report.

import_export Export Issues You can export issues to popular bugtrackers like Atlassian JIRA or use secure way to share only issues. Using JSON format, you can write your own integration into any system, like LaTeX, MS docs etc...

account_circle AI Integration You can add you private local LLM, ask AI to generate report summary and much more!
photo_library   Youtube channel Video tutorials to watch!
